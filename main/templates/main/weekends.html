Σαββατοκύριακο Καινοτομίας στις Αστικές Μεταφορές

28 - 30 Σεπτεμβρίου 2018

Έχετε μια ιδέα σχετικά με τις βιώσιμες αστικές μεταφορές που θέλετε να προωθήσετε ή ένα πρόβλημα που θέλετε να λύσετε, αλλά δεν ξέρετε από πού να ξεκινήσετε ή από ποιον να ξεκινήσετε;

Το Σαββατοκύριακο Καινοτομίας στις Αστικές Μεταφορές (ΣΚΑΜ) είναι το μέρος για να αναζητήσετε μια ομάδα, να δημιουργήσετε το πρωτότυπο της ιδέας σας, να επικυρώσετε την επιχειρηματική σας ιδέα και να λάβετε πληροφορίες από έμπειρους ανθρώπους που προέρχονται από το χώρο των μεταφορών και των επιχειρήσεων, όλα σε ένα Σαββατοκύριακο.

Τι πρέπει να φέρετε

Πολλή ενέργεια! Θα θέσετε τη δική σας ιδέα ή θα ακούσετε άλλους, κατόπιν θα ψηφίσετε τις πιο ενδιαφέρουσες ιδέες και θα διαμορφώσετε ομάδες με διαφορετικά σύνολα δεξιοτήτων. Μπορείτε να έρθετε με φίλους σας, αλλά καλύτερα να συμμετέχετε σε ξεχωριστές ομάδες. Θυμηθείτε, τα πάντα είναι θέμα ομάδας.

Η ιδέα που βάζετε και το πρόβλημα που θέτετε για επίλυση θα πρέπει να καλύψει θέματα σχετικά με τις αστικές μεταφορές και την χρήση νέων τεχνολογιών σε αυτές. Κατά τη διάρκεια του Σαββατοκύριακου, θα κληθείτε να δημιουργήσετε ένα πρωτότυπο δηλαδή ένα ελάχιστο βιώσιμο προϊόν που θα ταιριάζει στις ανάγκες του πελάτη-στόχου σας. Θα λάβετε καθοδήγηση κα συμβουλές, θα επαναπροσδιορίσετε την ιδέα σας και πιθανόν να διαμορφώσετε μια νέα προσέγγισή εξ ολοκλήρου!

Το Σαββατοκύριακο Καινοτομίας στις Αστικές Μεταφορές είναι μια μοναδική εμπειρία που περιλαμβάνει:

  • 7 πλήρη (και νόστιμα) γεύματα κατά τη διάρκεια του Σαββατοκύριακου
  • Επικοινωνία με καταπληκτικούς συμβούλους
  • Ένα νέο δίκτυο ανθρώπων προερχόμενων από τον χώρο των μεταφορών, προγραμματιστές, σχεδιαστές και επιχειρηματίες πρόθυμοι, όπως εσείς, να αλλάξουν τον κόσμο.
  • Όλο το διαδίκτυο και τον καφέ που μπορείτε να καταναλώσετε.

Αν έχετε ερωτήσεις, μη διστάσετε να επικοινωνήσετε μαζί μας στο REFERENCE EMAIL

Μην περιμένετε πια και δηλώστε συμμετοχή στο Σαββατοκύριακο Καινοτομίας στις Αστικές Μεταφορές!

Καμία άλλη συζήτηση, τώρα μόνο δράση!
